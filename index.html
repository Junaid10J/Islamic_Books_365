GitHub Settings

🔗 How GitHub Sync Works

When you upload a PDF, it is committed directly to your GitHub repository junaid10j/Islamic_Books_360 using the GitHub API. Your books are then accessible from any device, anywhere in the world.
📁 PDFs are saved to: books/your-book-title.pdf
📋 Book index saved to: books/books.json
✍️ Blog posts saved to: blog/posts.json
🌍 Accessible at: https://junaid10j.github.io/Islamic_Books_360/books/

🛠️ Setup Instructions

Follow these steps once to connect your GitHub repository:
1
Go to github.com/settings/tokens/new (GitHub → Settings → Developer settings → Personal access tokens → Tokens classic)
2
Give it a name like "Islamic Books 360". Under Scopes, check only repo (full control of private repositories). Click Generate token.
3
Copy the token (starts with ghp_) and paste it below. Store it safely — GitHub only shows it once.
4
Click Save & Test Connection. That's it — all future uploads will automatically commit to GitHub!
🔒 Your token is stored only in this browser's localStorage. It is never sent anywhere except directly to api.github.com.
